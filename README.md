# CLI Tools

A collection of simple Python command-line tools.

## Scripts

### folder_scanner.py
Scans a folder and outputs a summary report.

**Usage:**
```
python3 folder_scanner.py <folder_path>
```

**Output includes:**
- Total number of files
- Total size
- Largest file
- File types breakdown
- Saves a `scan_report.txt` in the scanned folder

---

### greeting.py
Asks the user for their name and prints a greeting.

**Usage:**
```
python3 greeting.py
```

## Requirements
- Python 3
